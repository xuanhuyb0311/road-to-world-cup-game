# ROAD TO WORLD CUP

## Giới thiệu

**ROAD TO WORLD CUP** là một trò chơi web lấy chủ đề bóng đá World Cup.  
Người chơi điều khiển nhân vật vượt qua các chướng ngại vật, ghi điểm và cố gắng đạt thành tích cao nhất.

Dự án được xây dựng bằng **HTML, CSS và JavaScript**, chạy trực tiếp trên trình duyệt mà không cần cài đặt server.

## Chức năng chính

- Giao diện bắt đầu game.
- Chọn nhân vật.
- Hiệu ứng loading.
- Điều khiển nhân vật trong quá trình chơi.
- Tính điểm.
- Hiển thị giao diện chơi lại khi thua.
- Trang Documentation giới thiệu và hướng dẫn chơi.

---

## Hướng dẫn chơi

1. Mở game tại `maingame/index.html`.
2. Nhấn nút **Play** để bắt đầu.
3. Điều khiển nhân vật vượt qua chướng ngại vật.
4. Cố gắng đạt điểm số cao nhất.
5. Khi thua, nhấn **Restart** để chơi lại.

## Hướng dẫn tải và sử dụng project

1. Vào repository GitHub.
2. Bấm nút **Code**.
3. Chọn **Download ZIP**.
4. Giải nén file tải về.
5. Mở thư mục project.
6. Vào `maingame/index.html` để chơi game.
